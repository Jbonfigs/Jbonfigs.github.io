A collection of courses taken during my first year of the PhD program, *Fall 2025 – Spring 2026*

EECE 5698: FPGAs in the Cloud
======
Covers heterogeneous compute elements — particularly FPGAs and GPUs — as cloud computing offerings. Topics include parallel architectures, High-Level Synthesis (HLS), custom data types for hardware accelerators, and security concerns for cloud computing. Assignments use the Open Cloud Testbed (OCT). Final project: FPGA-accelerated Monte Carlo simulation for Heston stochastic volatility option pricing.

*Instructor: Prof. Miriam Leeser*

EECE 7352: Computer Architecture
======
Graduate-level treatment of computer architecture covering processor pipelines, memory hierarchies, instruction-level parallelism, multi-core and many-core processors, memory coherency and consistency, virtualization, I/O, embedded systems, and low-power techniques. Uses Hennessy & Patterson as the primary text. Final project: fault injection attack on AES-GCM authenticated encryption.

*Instructor: Prof. David R. Kaeli*

EECE 7397: Advanced Machine Learning
======
Advanced topics in machine learning with a probabilistic focus. Covers Bayesian inference, graphical models, latent variable models, variational inference, Gaussian processes, generative models including VAEs, GANs, diffusion models, and normalizing flows. Students read and present research papers and complete an independent research project.

*Instructor: Prof. Jennifer G. Dy*

EECE 7398: Deep Learning for Embedded Systems
======
Graduate-level and research-oriented course covering recent advances in deep learning and its deployment on embedded hardware. Topics include deep CNNs, transformer-based models (BERT, ViT), LLMs, GANs, diffusion models, hardware acceleration of deep learning, and security aspects of deep learning systems. Programming in Python and C++ using PyTorch and TensorFlow.

*Instructor: Prof. Yanzhi Wang*
